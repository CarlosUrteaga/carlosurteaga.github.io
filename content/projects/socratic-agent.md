The Socratic Agent is a tutoring architecture designed to support learning without exposing model chain-of-thought or encouraging cognitive offloading.

The design uses an Elicit -> Structure -> Test -> Summarize loop, readiness gating, constrained dialogue acts, and curated tools for checking learner hypotheses.
